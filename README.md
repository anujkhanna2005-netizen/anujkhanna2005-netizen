## ◈ Featured Projects

<details>
<summary><b>⬡ AttendWise — College Attendance Tracker PWA</b></summary>

<br/>

A **Progressive Web App** engineered to solve a real problem for college students — tracking class attendance and maintaining the mandatory 75% threshold across all subjects with precision.

| Attribute | Details |
|:---|:---|
| **Stack** | TypeScript · React · Vite · Tailwind CSS · Firebase |
| **Type** | Progressive Web App (installable, offline-capable) |
| **Performance** | Offline-first architecture with Service Worker caching |
| **Security** | Firebase Auth with persistent session management |
| **Impact** | Actively used by college peers for attendance management |
| **Repository** | [github.com/anujkhanna2005-netizen/AttendWise](https://github.com/anujkhanna2005-netizen/AttendWise) |

AttendWise was built with a mobile-first approach, designed to be installed on a student's home screen like a native app. The architecture leverages Service Workers for offline resilience, ensuring students can view and update their attendance even without connectivity. The 75% threshold logic is computed dynamically per subject, giving students real-time insight into how many classes they can afford to skip or must attend.

<br/>
</details>

<details>
<summary><b>⬡ Expense Tracker PWA — Personal Finance Manager</b></summary>

<br/>

A **full-featured expense tracking Progressive Web App** built to deliver a native-grade personal finance experience entirely in the browser, with zero dependency on an internet connection post-install.

| Attribute | Details |
|:---|:---|
| **Stack** | TypeScript · React · Vite · IndexedDB / LocalStorage |
| **Type** | Progressive Web App (installable, fully offline) |
| **Performance** | Client-side persistence; zero-latency reads and writes |
| **Security** | All data stored locally — no server, no data leakage |
| **Impact** | Privacy-first finance tracking with offline capability |
| **Repository** | [github.com/anujkhanna2005-netizen/expense-tracker-pwa](https://github.com/anujkhanna2005-netizen/expense-tracker-pwa) |

Built with a privacy-first philosophy — financial data never leaves the device. The app uses client-side storage strategies to persist expense entries across sessions. Category-wise breakdowns and monthly summaries are computed on the client, making the experience fast and frictionless. The PWA manifest and service worker ensure full installability across Android and desktop browsers.

<br/>
</details>
